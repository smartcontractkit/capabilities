# Chainlink Local (CLL)

A CLI to create and run local nodes and nodesets.

```
NAME:
   cll - Run capabilities in a local environment

USAGE:
   cll [global options] command [command options] [arguments...]

COMMANDS:
   capabilities  Commands to manage the capabilities of the local nodes
   chain         Commands to manage the local chain
   client        Commands to manage the local chainlink client (binary)
   contracts     Commands to manage the EVM contract deployments
   jobs          Commands to manage the jobs of the local nodes
   node          Commands to manage the local nodes
   help, h       Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h  show help
```
